# 如何编辑并发布文章

这个主页部署在 GitHub Pages 上。访客只能浏览公开页面；只有你登录 GitHub 并拥有仓库权限时，才能新增、编辑、提交文章。

## 新增文章

1. 打开主页上的“写文章”入口，或进入仓库的 `_posts` 文件夹。
2. 新建一个 Markdown 文件，文件名建议使用 `YYYY-MM-DD-title.md`。
3. 在文件顶部保留下面这种信息：

```md
---
title: "文章标题"
date: "2026-07-15"
tags: ["note"]
---
```

4. 在下面写正文，然后点击 GitHub 的 `Commit changes`。

## 发布

提交后，GitHub Pages 会从仓库读取最新文件并更新主页。第一次启用 Pages 时，需要在仓库设置里选择从 `main` 分支的根目录发布。

## 后续增强

现在这版先保证可访问和可编辑。之后可以继续升级成自动读取 `_posts` 的完整博客，或者接入更像后台的 CMS。
